def add_noise(image, intensity=0.05):
    # Generate random pixel values using NumPy
    np_img = np.array(image)
    noise = np.random.randint(-255*intensity, 255*intensity, np_img.shape, dtype='int16')
    
    # Add noise to each pixel (RGB)
    noisy_img = np.clip(np_img.astype('int16') + noise, 0, 255).astype('uint8')
    return Image.fromarray(noisy_img)
What This Does:

Converts the solid black background (#1a1a1a) into a NumPy array
Generates random noise values between -20 and +20 for each RGB pixel
Adds the noise to the original color (e.g., 26 + 15 = 41, 26 - 10 = 16)
Clamps values to 0-255 to prevent overflow
Result: A subtly textured, grainy background (like film grain)
Answer: This is code-generated using the NumPy library. No assets are loaded. It creates a professional "film grain" effect on the fly.
